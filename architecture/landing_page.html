Proyecto Técnico Integrador: Ecommerce de Libros

Backend funcional en Java Spring Boot & MySQL

1. Descripción del Proyecto

Este proyecto académico consiste en el desarrollo de un sistema ecommerce para la venta de libros, con funcionalidades básicas de CRUD y gestión de usuarios, catálogo y pedidos. El backend está construido en Java con Spring Boot y utiliza MySQL como base de datos.

User Module Catalog Module Order Module Common Module

2. Entidades Principales

  • User: userID, nombre, email, password, enabled, createdAt
  • Role: roleID, nombre
  • Address: addressID, calle, ciudad, pais, codigoPostal, userID
  • Category: categoryID, nombre, descripcion
  • Book: bookID, titulo, autor, precio, stock, categoryID
  • Review: reviewID, contenido, rating, userID, bookID
  • Cart: cartID, userID, createdAt
  • CartItem: cartItemID, cantidad, bookID, cartID
  • Order: orderID, fecha, total, estado, userID
  • OrderItem: orderItemID, cantidad, precioUnitario, bookID, orderID
